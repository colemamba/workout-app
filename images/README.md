# Image Assets

Drop your Midjourney images here with these names:

## Required Images

| Filename | Description |
|----------|-------------|
| `body-base-front.png` | Blue translucent body, front view, no highlights |
| `body-base-back.png` | Blue translucent body, back view, no highlights |
| `body-push-front.png` | Front view with chest, front delts, triceps glowing red |
| `body-pull-back.png` | Back view with lats, traps, rear delts glowing red |
| `body-legs-front.png` | Front view with quads, calves glowing red |
| `body-legs-back.png` | Back view with hamstrings, glutes, calves glowing red |

## Optional (for more granular control)

| Filename | Description |
|----------|-------------|
| `body-chest-front.png` | Chest only highlighted |
| `body-shoulders-front.png` | All deltoids highlighted |
| `body-arms-front.png` | Biceps + triceps highlighted |
| `body-back-back.png` | Back muscles only highlighted |
